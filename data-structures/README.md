# 🧮 Estrutura de Dados (C/C++)

Implementações de algoritmos e estruturas de dados da disciplina **Estrutura de Dados (UFMS)**, em C e C++.

## 📦 Conteúdo
| Arquivo | Descrição |
|---|---|
| `altura.c` | Cálculo de altura (ex.: de árvore / estrutura). |
| `TrabalhoEstDados.cpp` | Trabalho principal da disciplina. |
| `Implementação com heap v1/EstDados01.cpp` | Implementação usando **heap**. |
| `3.4.txt` | Anotações/entrada do exercício. |

## ▶️ Como compilar e rodar
```bash
# C
gcc altura.c -o altura && ./altura

# C++
g++ TrabalhoEstDados.cpp -o trabalho && ./trabalho
g++ "Implementação com heap v1/EstDados01.cpp" -o heap && ./heap
```

C · C++ (compiladores gcc/g++)
