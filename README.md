# UFMS Studies

This repository is where I keep the coursework and academic projects from my BSc in Information Systems at UFMS (Computer Science / Information Systems). Instead of spreading everything across scattered "bucket" repos, I grouped the work by area. Each subfolder is a self-contained project with its own README and instructions.

It is an academic portfolio. The code here is my own. Client and professional projects live in their own dedicated repositories.

## Projects

| Folder | Area | What it is |
|---|---|---|
| [`data-structures/`](data-structures) | Algorithms and Data Structures | Trees, heaps and sorting in C/C++. Roadmap: grow into a small executable application built on these structures. |
| [`database-lab/`](database-lab) | Databases | SQL modeling (DDL/DML, queries, triggers) with PostgreSQL and MySQL via Docker. Roadmap: elevate from coursework into a complete, improved database project. |
| [`distributed-computing/`](distributed-computing) | Distributed Systems | A Flask MVC web service (Flask-Admin, SQLAlchemy, HTTP Basic Auth) running on Docker. |
| [`web-programming/`](web-programming) | Web | Front-end exercises (HTML/CSS/JS) plus a small React app. |
| [`quiz-app/`](quiz-app) | Web | A multi-page quiz platform (create / play / ranking) in vanilla HTML/CSS/JS. |

## Tech across the projects

C, C++, SQL (PostgreSQL / MySQL), Docker, Python (Flask), JavaScript, React, HTML/CSS.

## Running a project

Each subfolder has its own `README.md` with setup and run instructions: compile commands for the C/C++ work, `docker compose` for the database and Flask projects, and a static server for the web ones.
